    About Hear Clear

    Hear Clear is a system that use to analyse any sound or music given by user.

    There are two ways to send the sound :

    • Record the sound at that moment.
    • Upload the sound file from the device.

    Then, user need to click on 'submit' button to starts send the sound to the server to do the analysing process. The output of analyzed will be including:

    • Length of Sound.
    • File Type.
    • File Size.
    • Maximum Detected Frequency.
